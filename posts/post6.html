Python数据分析入门：用Pandas做简单的数据统计

发布日期：2024-04-20 | 标签：#Python #数据分析 #Pandas

大家好！我是Mingxi，一名对编程和数据分析充满热情的小学生。今天，我想和大家分享如何使用Python的Pandas库来进行简单的数据分析。数据分析是一种非常有用的技能，它可以帮助我们从大量的数据中发现规律和趋势。

什么是Pandas？

Pandas是Python中一个非常流行的数据分析库，它提供了强大的数据结构和数据分析工具。Pandas的名字来源于"Panel Data"（面板数据）和"Python Data Analysis"（Python数据分析）的组合。使用Pandas，我们可以轻松地读取、处理、分析和可视化数据。

准备工作

在开始之前，我们需要安装Pandas库。如果你还没有安装，可以请爸爸妈妈帮忙，在命令提示符中输入以下命令：

pip install pandas

安装完成后，我们就可以在Python程序中导入Pandas库了。

创建我们的第一个数据集

为了学习Pandas，我们可以先创建一个简单的数据集。假设我们对班级同学的考试成绩感兴趣，我们可以创建一个包含姓名、数学成绩、语文成绩和英语成绩的数据集。

# 导入pandas库，通常用pd作为别名
import pandas as pd

# 创建一个字典，包含学生的姓名和成绩
data = {
    '姓名': ['小明', '小红', '小刚', '小丽', '小强'],
    '数学': [85, 92, 78, 90, 88],
    '语文': [90, 88, 92, 85, 80],
    '英语': [82, 95, 88, 86, 90]
}

# 使用字典创建DataFrame，DataFrame是Pandas中最常用的数据结构
students_df = pd.DataFrame(data)

# 打印DataFrame，查看数据
print(students_df)

运行上面的代码，我们应该能看到如下的输出：

   姓名  数学  语文  英语
0  小明  85  90  82
1  小红  92  88  95
2  小刚  78  92  88
3  小丽  90  85  86
4  小强  88  80  90

基本的数据统计

现在，让我们来做一些简单的数据统计。Pandas提供了很多方便的函数，可以帮助我们快速计算数据的统计信息。

1. 计算平均分

我们可以使用mean()函数来计算每个科目的平均分：

# 计算每个科目的平均分
average_scores = students_df.mean()

print("各科平均分：")
print(average_scores)

运行上面的代码，我们应该能看到各科目的平均分。

2. 计算最高分和最低分

我们可以使用max()和min()函数来计算每个科目的最高分和最低分：

# 计算每个科目的最高分
highest_scores = students_df.max()

# 计算每个科目的最低分
lowest_scores = students_df.min()

print("各科最高分：")
print(highest_scores)

print("各科最低分：")
print(lowest_scores)

3. 计算总分

我们可以添加一个新的列，用来存储每个学生的总分：

# 计算每个学生的总分
students_df['总分'] = students_df['数学'] + students_df['语文'] + students_df['英语']

print("添加总分后的学生成绩：")
print(students_df)

运行上面的代码，我们应该能看到一个包含总分的新DataFrame。

4. 排序

我们可以使用sort_values()函数来根据总分对学生进行排序：

# 按照总分从高到低排序
sorted_students = students_df.sort_values(by='总分', ascending=False)

print("按总分排序后的学生成绩：")
print(sorted_students)

读取外部数据

除了创建自己的数据外，我们还可以使用Pandas来读取外部数据文件。Pandas支持很多种数据格式，包括CSV、Excel、JSON等。

假设我们有一个名为"scores.csv"的CSV文件，包含了学生的成绩信息。我们可以使用read_csv()函数来读取这个文件：

# 读取CSV文件
df = pd.read_csv('scores.csv')

# 打印前5行数据
print(df.head())

如果是Excel文件，我们可以使用read_excel()函数：

# 读取Excel文件
df = pd.read_excel('scores.xlsx')

# 打印前5行数据
print(df.head())

简单的数据可视化

数据可视化是数据分析的重要部分，它可以帮助我们更直观地理解数据。Pandas可以和Matplotlib库一起使用，来创建各种图表。

首先，我们需要安装Matplotlib库：

pip install matplotlib

然后，我们可以创建一个简单的条形图，来比较每个学生的各科成绩：

# 导入matplotlib库
import matplotlib.pyplot as plt

# 设置中文显示
plt.rcParams["font.family"] = ["SimHei", "WenQuanYi Micro Hei", "Heiti TC"]

# 创建一个条形图，比较每个学生的各科成绩
students_df.plot(kind='bar', x='姓名', y=['数学', '语文', '英语'], figsize=(10, 6))

# 添加标题和标签
plt.title('学生各科成绩比较')
plt.xlabel('姓名')
plt.ylabel('分数')

# 显示图表
plt.show()

运行上面的代码，我们应该能看到一个条形图，显示每个学生的各科成绩。

我的数据分析小项目

最近，我用Pandas做了一个小项目，分析了我家最近一个月的家庭支出情况。我收集了每天的支出数据，包括食品、交通、娱乐等方面的支出，然后用Pandas进行了分析。

通过这个项目，我发现：

  • 我们家在食品方面的支出最多，占总支出的40%左右。
  • 周末的支出通常比工作日多，尤其是在娱乐方面。
  • 这个月的总支出比上个月减少了10%，这可能是因为我们减少了外出就餐的次数。

通过数据分析，我们可以更好地了解自己的消费习惯，从而做出更明智的消费决策。

学习数据分析的小建议

作为一名小学生，我想分享一些学习数据分析的小建议：

  1. 从简单的项目开始：不要一开始就尝试复杂的数据分析项目，从简单的小项目开始，比如分析班级的考试成绩、自己的零花钱支出等。
  2. 多练习：数据分析是一项实践性很强的技能，只有通过不断地练习，才能真正掌握它。
  3. 学习基础的数学知识：数据分析需要一些基础的数学知识，比如统计、概率等。学习这些知识可以帮助你更好地理解数据分析的原理。
  4. 使用真实的数据：尽量使用真实的数据进行分析，这样可以让你的学习更有意义。你可以使用公开的数据集，或者收集自己感兴趣的数据。
  5. 可视化你的结果：使用图表来可视化你的分析结果，这样可以让结果更加直观和容易理解。
